# Sync APIs

Sync APIs é um projeto para **consumir dados de duas APIs externas** e armazená-los em bancos de dados PostgreSQL, permitindo que esses dados sejam utilizados posteriormente em outros projetos ou integrações.

## Funcionalidades

- Consome duas APIs externas (detalhes nos arquivos de configuração).
- Armazena dados recebidos em bancos PostgreSQL.
- Estrutura preparada para consumo por outros sistemas.
- Facilita integração entre sistemas internos e externos.
